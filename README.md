
# Library Database System

* A scalable library management system built with Python, Tkinter, and SQL, designed to efficiently manage book inventory and user records through an intuitive graphical interface
* Designed and implemented a full CRUD-based system for managing library data
* Improved query efficiency to handle larger datasets with reduced execution time
* Integrated frontend (Tkinter) with backend database operations

## Key Features

* Interactive GUI built with Tkinter for seamless user experience
* Relational database design for structured storage of books and users
* Borrow and return system with logical state handling
* Optimized SQL queries to improve performance for 200+ records

## Tech Stack

* Python
* Tkinter (GUI)
* SQL (Database Management)
